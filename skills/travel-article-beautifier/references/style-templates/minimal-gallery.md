# Style Template: Minimal Gallery

- Version: 0.1.0
- Last updated: 2026-07-14

Use this prompt template when the source is photo-led, image-caption-heavy, or the user wants a quiet visual essay.

## Recognition Signals

- Photos, captions, short notes, gallery, visual diary, hotel room, landscape, museum, island, architecture
- User asks for minimal, cinematic, clean, gallery, visual, or photo-led style
- Source has fewer words but strong visual moments
- The article should support images rather than overwhelm them

## Prompt

You are beautifying a travel article in a minimal-gallery style. Treat images and captions as the primary structure. Use restrained prose, precise captions, and generous pauses.

Write like a quiet visual essay:

- Start with a short mood-setting paragraph.
- Let each image or scene carry one clear idea.
- Keep captions specific and factual.
- Use brief prose bridges between image groups.
- Avoid heavy explanation unless the user asks for a guide.
- End with a small closing note rather than a grand conclusion.

Do not invent what appears in a photo. If images are not available, create placeholders that describe what image would fit, clearly marked as suggestions.

## Output Shape

- Minimal title options
- Visual sequence
- Caption set
- Short article prose
- Suggested image placeholders
- Missing image notes
